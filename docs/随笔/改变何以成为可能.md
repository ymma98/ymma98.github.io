# 改变何以成为可能

如果静静地观察自我，就会发现自身处于或大或小的轮回之中。心愿萌生之后，有计划，有行动，但结果不如意之事十之八九。心理状态也是，所谓闻誉则喜，闻毁则戚，内心似乎总比想象中更脆弱。更多的时候，分不清坚持和执拗的区别，傲慢与自信的区别，自卑与务实的区别。似乎清醒永远是短暂的，清冽的心境只会在意志耗尽、昏昏沉沉的状态持续很久之后短暂地出现。回望来时的路，仿佛沿着河水中央的窄路行走，前方是迷雾，两边是旋涡，或者在动摇中前行，或者一不小心就卷入水中，醒来时已回到起点。

我想这就是命理学得以成立的基础。

道家思想中有关于性和命的理解，性是超越了现象的理性与意志，是更抽象的宇宙意志的侧影。人与万物是宇宙的一部分，内在也都包含了宇宙意志的碎片，体现为个体的自性。命是超越个体的命运，不因个体的愿力发生改变。当人降生的瞬间，性命就在个体身上具象化了。此时心识开始发挥作用，人开始接触经验的世界。心是个体情感的所在，五官从经验世界获得的感受会唤起情感的跃动，当情感的潮水退去，自性就会显现。心之上，分别是魂和魄主导人的意识。魂代表意识中超然的理性，以及自性的一面；魄代表意识中欲望与经验的一面。魂属阳，魄属阴，魂是元神的具象，魄是识神的具象。识神以经验世界的见闻为养料，辅以人先天的生理与心理特质，代表着一个人的性格与智力、后天积累的知识与见闻、社会凝聚的集体潜意识等等。大部分人是由识神主导的。即，行动由欲望或者恐惧驱动，性格决定了一个人会采取怎样的抉择。

命分为天命和俗命。天命是由宇宙、自性、元神、超越性的魂等驱动的命，俗命是由识神、经验性的魄驱动的命。天命不可改，俗命可改，命理学计算的就是俗命。命理学假设，人出生那一刻的时空结构，决定了一个人的命运走向。人命运的走向和性格息息相关，而同一时空结构出生的人具有类似的性格，通过年月日时的周期性排列，命运的组合一共有五十余万种，可以根据八字或者紫微斗数等方法查表得到。「性格决定命运」其实是命理学成立的前提之一。一个人如果由识神主导，那么归根到底，他始终在实践识神的意志，这是一种性格裹挟着欲望的产物，佛家称此为「小我」，由小我驱使做出的行为即为「业力」。识神或小我张成了一个惯性极强的系统，而陷入该系统的人只能昏沉地做着自动化的反应，沉浸在各种欲求与痛苦之中。在这样业力系统下画出的人生轨迹就是俗命，不论是驱动俗命背后的因，还是导致俗命的果，佛家都称之为「相」。只要人一日沉浸在相之中而不自知，人就没有改命的可能。人们不断地算命、拜佛、祈福，
