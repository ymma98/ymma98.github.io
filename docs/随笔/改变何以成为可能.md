# 改变何以成为可能

如果静静地观察自我，就会发现自身处于或大或小的轮回之中。心愿萌生之后，有计划，有行动，但结果不如意之事十之八九。心理状态也是，所谓闻誉则喜，闻毁则戚，内心似乎总比想象中更脆弱。更多的时候，分不清坚持和执拗的区别，傲慢与自信的区别，自卑与务实的区别。似乎清醒永远是短暂的，清冽的心境只会在意志耗尽、昏昏沉沉的状态持续很久之后短暂地出现。回望来时的路，仿佛沿着河水中央的窄路行走，前方是迷雾，两边是旋涡，或者在动摇中前行，或者一不小心就卷入水中，醒来时已回到起点。

我想这就是命理学得以成立的基础。

道家思想中有关于性和命的理解，性是超越了现象的理性与意志，是更抽象的宇宙意志的侧影。人与万物是宇宙的一部分，内在也都包含了宇宙意志的碎片，体现为个体的自性。命是超越个体的命运，不因个体的愿力发生改变。当人降生的瞬间，性命就在个体身上具象化了。此时心识开始发挥作用，人开始接触经验的世界。心是个体情感的所在，五官从经验世界获得的感受会唤起情感的跃动，当情感的潮水退去，自性就会显现。心之上，分别是魂和魄主导人的意识。魂代表意识中超然的理性，以及自性的一面；魄代表意识中欲望或情欲的一面。魂属阳，魄属阴，魂是元神的具象，魄是识神的具象。识神以经验世界的见闻为养料，辅以人先天的生理与心理特质，代表着一个人的性格与智力、后天积累的知识与见闻、社会凝聚的集体潜意识等等。大部分人是由识神主导的。即，行动由欲望或者恐惧驱动，性格决定了一个人会采取怎样的抉择。

命理学假设，人出生那一刻的时空结构，决定了一个人的命运走向。人命运的走向和性格息息相关，而同一时空结构出生的人具有类似的性格，这样的命运组合一共有五十余万种。「性格决定命运」其实是命理学成立的前提之一。
